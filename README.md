# IT Recruitment System

A command-line recruitment tool that analyzes candidate profiles and recommends hiring decisions.

---

## How it works
Each candidate is evaluated based on:
- **Technical score** — calculated from test score, GitHub projects, and experience
- **Affordability** — salary expectation under 6000€
- **Communication** — English level and test score

---

## Decisions
- **Hire** — strong technical score, affordable, good communication
- **Interview** — strong technical score and communication, but over budget
- **Reject** — does not meet minimum requirements

---

## How to run
```bash
python main.py
```
